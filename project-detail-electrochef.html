ElectroChef

전자공학 학부생을 위한 웹 기반 계산 도구

기간

2025.09. - 2025.10.

역할

개인 프로젝트 (기획, 개발, 디자인)

기술 스택

HTML TailwindCSS JavaScript

프로젝트 배경

ElectroChef는 GCHQ의 CyberChef에서 영감을 받아 시작된 개인 프로젝트입니다. CyberChef가 보안 분야의 데이터 변환 및 분석 작업을 웹에서 편리하게 수행하도록 돕는 것처럼, 전자공학 분야에서 자주 사용되는 복잡한 계산들을 웹 브라우저에서 손쉽게 처리할 수 있는 도구를 만들고 싶었습니다. 전공 과제를 하거나 시험공부를 할 때마다 매번 공학용 계산기를 찾거나 복잡한 수식을 다시 유도해야 했던 불편함을 해결하고자 했습니다.

주요 기능

ElectroChef는 전자공학 학부 과정에서 가장 빈번하게 사용되는 핵심 변환 기능들을 제공하여 학습과 과제 해결의 효율을 높입니다.

  • 실시간 각도 변환: 라디안(Radian) 값을 입력하면 즉시 도(Degree) 값으로, 도 값을 입력하면 라디안 값으로 실시간 변환됩니다.
  • 페이저(Phasor) 변환: 회로이론의 교류(AC) 해석에 필수적인 페이저 변환을 지원합니다. 직교좌표 형식(a + jb)을 극좌표 형식(r∠θ°)으로, 또는 그 반대로 변환하는 기능을 제공하여 복소수 계산의 편의성을 크게 높였습니다.

성과 및 배운 점

이 프로젝트를 통해 바닐라 JavaScript와 TailwindCSS만으로 사용자 인터랙션을 구현하고 깔끔한 UI를 구성하는 경험을 할 수 있었습니다. 특히, 사용자의 입력을 실시간으로 감지하여 결과를 즉시 업데이트하는 로직을 구현하며 JavaScript의 이벤트 처리와 DOM 조작에 대한 이해를 심화시킬 수 있었습니다. 향후 스미스 차트(Smith Chart) 계산, 임피던스(Impedance) 변환 등 더 다양한 기능을 추가하여 발전시켜 나갈 계획입니다.
