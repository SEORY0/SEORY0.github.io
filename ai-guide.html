Part 0 — AI 도구를 제대로 쓰기 위한 기본기

#LLM #프롬프트엔지니어링 #모델비교

LLM의 동작 원리를 개발자 눈높이로 짚고, 프롬프트 엔지니어링 핵심 패턴(System Prompt, Few-shot, CoT, Tool Use)을 정리합니다. GPT-4o, Claude, Gemini, 로컬 모델의 특성 비교와 API / Chat UI / IDE 통합 중 상황별 선택 기준까지.

준비 중입니다.

Part 1 — 반복 작업을 AI에게 넘기는 법

#생산성 #검색 #요약 #자동화

Perplexity·ChatGPT Search·Claude Web Search 같은 AI 검색 도구 비교, 글쓰기 보조 워크플로우, 논문·영상·회의록 자동 요약 파이프라인, 그리고 Zapier AI·Make·n8n과 LLM을 연동한 자동화 사례를 다룹니다.

준비 중입니다.

Part 2 — AI와 함께 코딩하기

#코딩어시스턴트 #Copilot #ClaudeCode #Cursor #MCP

Copilot, Cursor, Claude Code, Windsurf 등 AI 코딩 어시스턴트를 비교하고, 코드 생성→리뷰→디버깅→리팩토링 실전 워크플로우를 소개합니다. README 작성, 커밋 메시지, PR 요약 자동화와 MCP 서버를 활용한 개발 도구-LLM 연결까지.

준비 중입니다.

Part 3 — 사이버보안 연구에 AI 끼얹기

#취약점분석 #위협인텔리전스 #RAG #CTF

LLM을 활용한 소스코드·바이너리 취약점 분석, CVE/ATT&CK 데이터 기반 RAG 파이프라인, 논문 리서치 워크플로우, 그리고 CTF 리버싱·포너블·크립토 문제에서의 AI 보조 전략을 정리합니다.

준비 중입니다.

Part 4 — 이미지·영상·음악 AI 실전 가이드

#이미지생성 #영상AI #음악AI #디자인

Midjourney, DALL-E, Flux 비교와 프롬프트 팁, Sora·Runway·Kling 등 영상 AI 현황, Suno·Udio 같은 음악 생성 도구를 소개합니다. 블로그 썸네일, 발표 자료, 프로토타입 목업 등 개발자 관점의 활용법 위주로.

준비 중입니다.

Part 5 — 소비자에서 빌더로: 나만의 AI 시스템 구축

#CustomGPTs #로컬LLM #RAG구축 #에이전트

Custom GPTs / Claude Projects / Gems 비교, Ollama·vLLM으로 로컬 LLM 세팅, 임베딩→벡터DB→검색으로 이어지는 RAG 시스템 직접 구축, LangGraph·CrewAI 등 에이전트 프레임워크까지. AI를 쓰는 사람에서 만드는 사람으로.
